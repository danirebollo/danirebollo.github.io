FAQ

Preguntas frecuentes

¿Seguro que no tengo que usar espacios comunitarios?

Seguro. OpenLock se instala dentro de tu propia vivienda, tanto la cerradura como el módulo telefonillo.

¿Debo realizar obra para instalar OpenLock?

OpenLock se compone de dos elementos: la cerradura y el módulo telefonillo. La cerradura se fija a la puerta con pegamento de montaje, cinta de doble cara o tornillos. Recomendamos usar tornillos en lugares donde se necesite una mayor robustez, como en casas de alquiler eventual o donde haya un gran tránsito, como en empresas.

¿Tengo que pagar una suscripción por usar OpenLock?

Si tu uso es particular no tienes que pagar por usar OpenLock. Si necesitas usar nuestra APP de gestión de visiendas o acceso a nuestra API contacta con nosotros para que podamos darte una cotización según tus necesidades

¿Qué conectividad tiene OpenLock?

La versión comercial de OpenLock tiene conectividad WiFi y Bluetooth. Sin embargo, si el lugar de instalación no tiene conectividad WiFi disponible puedes solicitar la versión extendida de OpenLock con SIM 3G/4G

¿Puedo abrir la puerta si no tengo internet en el teléfono?

Por supuesto. Es frecuente en viviendas de alquiler vacacional que el inquilino, por lo menos el primer día, no tenga internet después del viaje. Por este motivo hemos preparado OpenLock para poder realizar aperturas mandando un SMS desde los números autorizados.
